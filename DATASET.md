# Dataset Information

## Name
CelebA (CelebFaces Attributes Dataset)

## Source
[Kaggle CelebA Dataset](https://www.kaggle.com/jessicali9530/celeba-dataset)
Also available at [MMLAB](http://mmlab.ie.cuhk.edu.hk/projects/CelebA.html)

## Description
A large-scale face attributes dataset with more than 200K celebrity images, each with 40 attribute annotations.
